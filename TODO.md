## Полная раскладка кнопок (DualSense)

### Мир (без открытых окон/меню)

| Кнопка | Действие |
|--------|----------|
| **Левый стик** | Движение персонажа |
| **Правый стик** | Мышь |
| **L2 + Правый стик** | Вращение камеры |
| **L1** | Левая кнопка мыши (LMB) |
| **R1 tap** | SmartClick (правый клик по ближайшему объекту) |
| **R1 hold** | Открыть RadialPicker (выбор цели) |
| **R2** | Открыть InterfaceMenu (Карта/Персонаж/Кин/Инвентарь/Экипировка) |
| **A / B / X / Y** | Hotbar слоты 1/2/3/4 |
| **L2 + A/B/X/Y** | Hotbar слоты 5/6/7/8 |
| **D-pad** | Навигация MenuGrid (крафт/эмоции/умения снизу справа) |
| **Select** | Подтвердить выбор в MenuGrid |
| **Start** | ESC (глобально) |
| **B** | Отменить режим MenuGrid |

### FlowerMenu (контекстное меню объекта)

| Кнопка | Действие |
|--------|----------|
| **Правый стик** | Выбор лепестка по углу |
| **D-pad** | Навигация между лепестками |
| **R1 / R2** | Подтвердить выбор |
| **B** | Отменить / закрыть |

### RadialPicker (выбор цели R1 hold)

| Кнопка | Действие |
|--------|----------|
| **Правый стик** | Выбор цели по углу |
| **R1 release** | Подтвердить |
| **B** | Отменить |

### InterfaceMenu (R2)

| Кнопка | Действие |
|--------|----------|
| **Правый стик / D-pad** | Выбор пункта |
| **R2** | Подтвердить |
| **B** | Отменить |

### Открытое окно (инвентарь, крафт, персонаж и т.д.)

| Кнопка | Действие |
|--------|----------|
| **D-pad** | Навигация по слотам инвентаря / прокрутка |
| **Select** | Взять предмет в курсор (LMB, pick up) |
| **A** | ENTER (крафт предмета, подтвердить диалог) |
| **L3** | Прокрутка вверх |
| **R3** | Прокрутка вниз |
| **Start** | Закрыть окно (reqclose) |
| **B** | Снять выделение слота → затем закрыть окно (ESC) |

## Реализовано ✓

- GamepadManager (jinput, DualSense, volatile snapshot)
- DirectMovement: левый стик → клики wdgmsg("click"), угол камеры, деадзона, maxClickWU
- SmartTarget: конус + ближний радиус, приоритеты, боевой режим, гейз из последнего движения
- RadialPicker: R1 hold открывает, RS навигация, R1 release подтверждает, B отменяет
- Камера: L2+RS вращает (все типы камер), RS-click сбрасывает угол
- ABXY → hotbar слоты 1-4 / 5-8 (L2 модификатор)
- L1 → LMB эмуляция
- HUD бейджи: ⌖ GAMEPAD + ⚔ COMBAT
- Боевой режим: автодетект из Fightview.lsrel
- Индикатор цели: курсор hand над гобом + название FlowerMenu-стилем, 200 мс grace period
- FlowerMenu: RS выбор лепестка, D-pad навигация, R1/R2 подтвердить, B отменить
- InterfaceMenu (R2): радиальное меню, D-pad/RS навигация, R2 подтвердить, B отменить
- D-pad приоритеты: FlowerMenu > InterfaceMenu > RadialPicker > FocusedWindow > MenuGrid
- MenuGrid: D-pad навигация (по строкам ←→, по столбцам ↑↓), Select подтверждает
- Окна: автофокус через gui.focused, D-pad навигирует инвентарь / скроллит, Select=LMB take, A=ENTER, B=отмена/закрыть, Start=закрыть
- L3/R3: прокрутка вверх/вниз (в окне и в мире)
- Start: закрыть окно (прямой reqclose) или ESC в мире
- Select: взять предмет LMB в инвентаре, подтвердить MenuGrid
